Function to get the value of the metrics usually employed in PAM minimization: sum of distances of each point to its closest medoid divided by number of points.

Parameters
[in]LmedA vector with the indices of the points which are medoids. These indices refer to the order of points in the distance/dissimilarity matrix
[in]LclasifA vector with the index (as position in Lmed) of the medoid closest to each point
[in]DA reference to the dissimilariry matrix, as a SymmetricMatrix
Returns
The value of the total sum of distances divided by the number of points
