unsigned int ChooseNumThreads ( int  nthreads)

Auxiliary function to decide the number of threads

Parameters
nthreadsNumber of thread the user wish to launch. It can be:
-1 to indicate not to use threads (strictly sequential computation)
0 to let the program choose according to the available number of cores/threads in the machine
Other possitive number to ask for such number of threads
Returns
The number of threads that will be really used. This is:
1 for input value -1
the number of cores/threads in the machine for input value 0 (if hypertheading exists, it is taken into account)
the chosen number for any other input value (but a warning message is raised if the chosen number is bigger than the number of cores/threads)
