\doxysubsection{Detailed Description}
\subsubsection*{template$<$typename disttype$>$\newline
class Fast\+PAM$<$ disttype $>$}

A class to implement the Partitioning Around Medoids (PAM) clustering metho described in~\newline
~\newline
Schubert, E. and Rousseeuw, P.\+J.\+: \char`\"{}\+Fast and eager k-\/medoids clustering\+: O(k) runtime improvement of the PAM, CLARA, and CLARANS algorithms.\char`\"{}~\newline
Information Systems, vol. 101, p. 101804, 2021.~\newline
doi\+: \href{https://doi.org/10.1016/j.is.2021.101804}{\texttt{ https\+://doi.\+org/10.\+1016/j.\+is.\+2021.\+101804}}~\newline
~\newline
Notice that the actual values of the vectors (instances) are not needed. To recover them, look at the data matrix used to generate the distance matrix.~\newline
The number of instances, N, is never passed since dissimilarity matrix is NxN and therefore its size indicates the N value.~\newline
With respect to the calculated value, it consists on two vectors. The first one has as many components as requested medoids and the second has as many components as instances.~\newline
Medoids are expressed in the first one by its number in the array of points (row in the dissimilarity matrix) starting at 0 (C++ convention).~\newline
The second vector contains the number of the medoid (i.\+e.\+: the cluster) to which each instance has been assigned, according to their order in the first vector (also from 0).~\newline
These vectors are returned by the functions Get\+Medoids and Get\+Assign (see their respective documentation)
