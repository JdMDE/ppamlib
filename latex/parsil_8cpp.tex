\doxysubsubsection*{parsil}

A program to calculate the silhouette of a clustering (usually obtained with parpam) in parallel

The program must be called as

parsil dissim\+\_\+file clasif\+\_\+file \mbox{[}-\/nt numthreads\mbox{]} -\/o out\+\_\+file\+\_\+name

where~\newline
~\newline
 {\bfseries{dissim\+\_\+file}}\+: File with the dissimilarity matrix in jmatrix format.~\newline
 It must be a Symmetric\+Matrix of float or double with dimension (n x n).~\newline
~\newline
 {\bfseries{clasif\+\_\+file}}\+: File with the clasification result, as obtained from program parpam.~\newline
 It must be a (n x 1) matrix (a column vector) of unsigned int values with values in 0..(k-\/1) being k the number of clusters.~\newline
~\newline
 {\bfseries{numthreads}}\+: Requested number of threads.~\newline
 Setting it to 0 will make the program to choose according to the number of processors/cores of your machine (default value).~\newline
 Setting to -\/1 forces serial implementation (no threads)~\newline
~\newline
 {\bfseries{out\+\_\+file\+\_\+name}}\+: Name of the file contaning the silhouette. Compulsory.~\newline
 The output file will be a Full\+Matrix of double type and dimension (n x 1) (a column vector) with the value of the silhouette for each point.~\newline
~\newline
 Points are assumed to be in the same order in the dissimilarity matrix and the clasification vector, and this is the order in which their~\newline
 silhouettes will be written in the output vector. If the matrix has row names, they will be set for the output file. It the clasif vector~\newline
 has row names, they will be checked against the row names of the matrix, if both are present. If only clasification vector has names,~\newline
 they will be set for the output vector.~\newline
 The program will refuse to load the dissimilarity matrix if not enough RAM is available; also, it will show a warning if the required amount~\newline
 of memory to load it is above 75\% of the available RAM.~\newline
 Remember that using the program \textquotesingle{}jmat csvdump ...\textquotesingle{} you can convert the output file to .csv format.~\newline
