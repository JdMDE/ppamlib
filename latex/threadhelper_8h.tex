Auxiliary function to decide the number of threads


\begin{DoxyParams}{Parameters}
{\em nthreads} & Number of thread the user wish to launch. It can be\+:~\newline
-\/1 to indicate not to use threads (strictly sequential computation)~\newline
0 to let the program choose according to the available number of cores/threads in the machine~\newline
Other possitive number to ask for such number of threads~\newline
 \\
\hline
\end{DoxyParams}
\begin{DoxyReturn}{Returns}
The number of threads that will be really used. This is\+:~\newline
1 for input value -\/1~\newline
the number of cores/threads in the machine for input value 0 (if hypertheading exists, it is taken into account)~\newline
the chosen number for any other input value (but a warning message is raised if the chosen number is bigger than the number of cores/threads) 
\end{DoxyReturn}
